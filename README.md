# vue2.0-vue-router2-axios-webpack
Vue.js 2.0开发脚手架，自带Vue-router2.0和axios

install package: 

```
npm install 或 nam i
```

dev:

```
 npm run dev
```

build:

```
 npm run build
```
### tip:###

2.0+的vue-cli的dist目录下的打包文件无法直接在本地已file://的形式打开，需要在服务器容器中打开：

```
1.cd dist
2.npm install -g http-server
3.fs
```
